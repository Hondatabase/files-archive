RAM

Location Bytes Description Notes
00CC 1 VSS Sensor km/h
0210.3 1b Power Steering Pressure(PSP) switch input If pin B8 (PSP) is grounded, 0x0210.3 will read 1

ROM:

Location Bytes Description Notes
0652 6 Injector Test Jump Routine #1/2 Change to "03 5F 06" to disable injector test - must apply both patches
1292 1 VTEC Coolant Temp Check (0x44 enables, 0xFF disables)
1580 6 Injector Test Jump Routine #2/2 Change to "03 9A 15" to disable injector test - must apply both patches
238A 6 O2 Heater Jump Routine Change to "03 C5 23" to disable O2 heater
2BAD 6 Checksum Jump Routine Change to "03 B6 2B" to disable (2BAD, no checksum for you ! ;)
445C 6 Vtec Solenoid Check Jump Routine Change to "03 7A 44" to disable VTS error only - UNTESTED

60FB 1 Debug Mode (0xFF enables, 0x00 disables)
