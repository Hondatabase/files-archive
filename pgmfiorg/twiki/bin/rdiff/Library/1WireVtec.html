%META:TOPICINFO{author="blundar" date="1077237440" format="1.0" version="1.1"}% The 1-wire vtec codebase is a ROM for the OBD0 MPFI ECUs that allows the Automatic Transmission Lockup Solenoid to be used to control vtec engagement. Jason Parker? came up with the Hardware For One Wire Vtec changes.

As of Oct 2 2003 it is functional but needs some help and further testing. Dave Blundell? has several cars running around on it. If you want to contribute/work/see it, check out the 1w_vtec code module from Source Forge
