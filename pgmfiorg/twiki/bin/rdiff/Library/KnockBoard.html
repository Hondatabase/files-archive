%META:TOPICINFO{author="tekphobia" date="1081742820" format="1.0" version="1.1"}% the knock board is a circuit board (seperate from main ECU PCB) mounted in the top-right hand corner of DOHC Vtec ECUs having a Knock Sensor. The Knock Board is responsible for processing the essentially auditory signal from the Knock Sensor and triggering an alarm if Pre Ignition is detected. In their stock form, the honda Knock Board might be useful for detecting knock from bad gas, although this is rather questionable.

Conventional wisdom says honda knock boards suck for detecting detonation while boosting. Conventional wisdom says honda knock boards freak over forged internals and pull timing during normal operation.

If you use a program that has the knock sensor disabled in an ECU that has a knock board present, the ECU will ignore the Knock Board's presence - in fact you can remove the Knock Board without any noticeable effect.
