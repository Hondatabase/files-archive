I posted the Options Close-up for the PK2-6640 manual B20A ECU.

I don't know if these options are related to the transmisson type or emissions. The auto PH3 ECU has the same options configuration as this manual ECU so I'm not sure what these do yet.

The PK2 and PH3 ECUs share almost the same PC board. Theya re 99% similar. So similar in fact that the PH3 code can run on the PK2. The BR8 Jumper seems to control the Secondary O2 sensor input. The PH3 doesn't have this sensor so it's deactivated by BR8. You can see the PH3 section for more details.
