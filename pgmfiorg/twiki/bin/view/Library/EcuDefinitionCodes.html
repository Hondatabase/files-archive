Honda ECU definition codes

These id codes are the last 3 digits printed directly on the MCU after the model number (e.g. M66207-251 - the "251" portion will be a much larger font). This code indicates the ECU program that is burned into the internal ROM of the MCU. ECUs with factory external EPROMs will not always have a code printed on the MCU if they do not have any internal ROM storage, as indicated by the "X" in their part number (e.g. M66X207)

ROMs for many of these are available, but if you have one not there, please post it up! You can upload bins using the "Attach" feature of this page. If you can't figure out the table, just add a written entry (or attachment) to the end of this and someone will incorporate it into the table for you.

The label on the ECU can often tell you a lot about it. See Label Decode for more information.

''Note'': Exactly 8 vertical bars on a line or you fook up the formatting :)
