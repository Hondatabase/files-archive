ppl are working on a ROM where the ECU activates the nitrous solenoid instead of the usual WOT switch. The main advantage of this is that the ECU knows when nitrous is active, and can therefore switch to a different set of fuel/timing maps. Dry kit + 450cc injectors + ECUNitrous Control w/ Dual Tables = FINE TUNED wet kit!
