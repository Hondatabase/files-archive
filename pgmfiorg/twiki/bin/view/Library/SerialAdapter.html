All Honda ECUs use 5v TTL signaling for serial communication. In order to connect to an ECU, if the hardware used for Data Logging doesn't support this, you must use some form of serial adapter. Below are some of the more common adapters available:

Unit From Interface To Interface
Super Droid Robots RS232 TTL
CompSys RS232 TTL
Lucas Schaar RS232 TTL
Gigatechnology USB 1.1 TTL
