Resources:

Beginners Section

(blundar sez: if you're getting started, read these in the order they appear. Much wisdom has been distilled into these pages.)
  • What You Need - to get started chipping ECUs
  • Introduction To ECUChipping - A great article to help you get started.
  • Chipping With Pictures - want to visually see what is involved in chipping an ECU?
  • Basic Concepts - "vocabulary" of ECUs.... read here if you want to understand what's going on
  • Understanding Maps - to help you understand and read fuel and ignition maps
  • Begginers FAQ - intended to help those who are getting started after the above resources.
  • Ecu Troubleshooting - how to figure out what isn't working

Tuning

  • Data Logging - Using outboard hardware connected to the ECU to monitor operation.
  • Tuning Concepts - The theory behind tuning a car properly
  • Ecu Tuning - Procedures for best tuning an ECU for your engine
  • Bin Repository - The Place to go if you need stock binaries.
  • Wide Band O2 - Used for accurately measuring Air/Fuel ratio
  • Tuning Timing - Without a guage like a Wide Band O2 to provide feedback, tuning ignition timing can be challenging
  • Real Time Programming - Used for tuning on the fly.
  • Why Do Piggybacks Suck - An opinion on why piggyback systems such as AFCs and E-Manage suck for tuning hondas

ECU Hardware/Software - how ECUs themselves work

  • Ecu Definition Codes - master list of ECUs, MCUs inside them and stock code dumps.
  • Ecu Trouble Codes - figure out what that CEL means!
  • ECUHardware - Everything you ever wanted to know about the hardware powering your ECU
  • Ecu Families - a heirarchal listing of ECUs grouped by similar design. (work in progress)
  • Ecu Sensors - Everything the ECU is connected to
  • Ecu Connections / Conversion - How everything is connected to the ECU. ECU Wire Harness dictionary.
  • Rom Maps - a list of information about addresses in ROM files: a map of what is found where

PGMFI.ORG-made Software - software written by forum members

  • ASM662 - Andy Sloane's open source Oki NX8/200 code 66K assember/disassembler suite
  • ECUControl - Nick Mailloux's program(s) to do Data Logging
  • Crome ROMEditor - John Cui's (mainly) OBD1 Civic Integra ROMEditor
  • Uber Data ERM - Blake's ROMEditor suporting OBD1 Civic Integra ECUs.
  • BRE - Ben's Rom Editor suports OBD0 Vtec ECUs, OBD0 MPFI ECUs, and some OBD1 Civic Integra ECUs
  • Ghetto Dyne - Mike Robichaud's OBD0 MPFI ROMEditor
  • Turbo Edit - Jason Parker's OBD0 MPFI ROMEditor
  • OBD0 Edit - Carsten's ROM Editor
  • Nico Analyze - Nico's "analyze.exe" used for finding tables and data in unknown ROMs.

Devices - hardware designed by PGMFI members

  • G-Spot - the DIY acceleration meter
  • Real Time Programmable Boards - boards to allow Real Time Programming
  • Ignitor Replacement - get rid of the ignitor in the distributor if you have a MSD or similar ignition amplifier
  • Mcu Readers - used to read copyprotected MCUs like the 83C154, 66207, 66507, 66509, 66589
  • Engine Sim - device to simulate a running motor (work in progress)
  • Knock Replacement - project to replace and improve the honda knock sensor board

Coding - resources for learning and modifying ECU software

  • Intel8051 - resources available for working with Oki 8051-based processors used in most OBD0 implementations
  • 66k Resources - resources available for working with Oki 66xxx processors used in most OBD1 and OBD2 implementations
