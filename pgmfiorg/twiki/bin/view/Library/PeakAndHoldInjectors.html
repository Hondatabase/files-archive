Peak And Hold Injectors aka Low Impedance Injectors.

Impedance typically ~2-5ohms. At 12volts, this is a draw of ~6-2.4 amps The high current will cause the injectors to run hotter.

The peak current opens the injector "crisply" and decisively. Then a lower "Hold" current keeps the injector open.

To run these injectors with the honda ecu, you may need to use an inline Resistor Box.

OBD0 already has a resistor box wired in. OBD1 does not have a Resistor Box. OBD2 does not have a Resistor Box.
