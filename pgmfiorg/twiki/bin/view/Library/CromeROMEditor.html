Crome is ROMEditor that John Cui? wrote. Its main feature that differentiates itself from other ROMEditors is that it supports the use of Java Script to control it and implement new features.

The Crome Script API allows you to extend the capabilites of the editor with relative ease. People have already written many scripts for Crome, most of which can be found on the PGMFI Source Forge page.

The Crome FAQ is a resource designed to get you started with Crome.

More information can be found at the Crome forum or at http://www.tunewithcrome.com/.
