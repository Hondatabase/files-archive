Aka "Limp Home Mode"

Limp Mode is the mode of operation when the ECU's CEL is lit solid red indicating a major malfunction. It is intended to get the car home, and will run the car very poorly. The backup processor is running the car without using ROM at this point. Only the TDCSensor? and Tps Sensor are used in this mode. Revlimit is encountered at 3500rpm.
