Обновление внутренней прошивки CobraRTP FlashOnline. Для обновления используйте файл .UPD соответствующий вашей ревизии платы Rev X.X (см. на плате устройства).
Инструкция в архиве с программой CobraRTP Utulity.

Изменения:

v1.25 bugfix
-Исправлено кратковременное прерывание связи с ЭБУ через 1 сек. после подачи питания (для 28F200, 29F200, 29F400).

v1.24 bugfix
-Исправлена неверная организация памяти при определенных условиях (блок памяти/тип памяти) 

v1.23 (необязательное)
-Добавлена поддержка фкнкции диагностики RAM эмулятора (CobraRTP Utility v2.97+)
-Исправлена ошибка индикации заряда батареи
-Добавлена возможность стирать память
-Незначительные доработки

v1.22
-Добавлена поддержка кодирования данных для ЭБУ с нестандартным подключением флэш памяти к процессору (Siemens 5WY (Simk31-43), 5WK (SMG2), MS5150, EMS2000, Sirius32, Simos 3 и прочие)
-Добавлен контроль заряда backup батареи устройства (только для Rev3.1 и выше)
-Оптимизирован код
(Для поддержки этих функций требуется CobraRTP Utility 2.9+)

v1.21
-Исправлены некоторые баги

v1.20
-Реализована функция сканер обращений (трассировка таблиц в TunerPRO RT)! (протестировано с Bosch ME7.1, EDC15P+, Siemens MS41/42/43)

v1.15b
-Исправлен баг, при котором автомобиль мог глохнуть при первом изменении памяти эмулятора
-Исправлено зависание эмулятора при выходе из TunerPRO или при программном отключении.
-Сделаны попытки для достижения работоспособности трассировки в TunerPRO
